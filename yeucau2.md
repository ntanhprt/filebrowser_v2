Thêm chức năng:
1. bấm 1 nút trên menu để mở pannel preview, bên phải
	- Khi mở preview, bấm vào file nào, thì file đó được hiển thị nội dung ở bên phải, nội dung phù hợp với định dạng, nếu là markdown thì hiển thị preview render
2. Giữ nguyên thống nhất định dạng xem: 
	- Khi user mở một file markdown để xem nội dung, ví dụ chuyển đổi sang xem dạng markdown đã render, thì lưu cái dạng xem này làm mặc định, bất kỳ file md nào cũng sẽ được hiển thị theo cài đặt này cho đến khi user chuyển đổi cách xem khác.
